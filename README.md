# SmartNotepad
Проект «SmartNotepad» представляет собой программное приложение типа текстового редактора с расширенными возможностями для работы с программным кодом. Основной особенностью системы является поддержка семантики языков программирования, наличие механизма синтаксической подсветки и возможность гибкой настройки тем оформления интерфейса.
