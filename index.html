01 — О проекте

Что такое SmartNotepad?

SmartNotepad — это современный текстовый редактор, разработанный в рамках дисциплины «Профессиональная практика программной инженерии». Проект демонстрирует применение принципов модульной архитектуры, командной разработки с использованием Git и документирования программного обеспечения.

Редактор ориентирован на разработчиков и студентов, которым необходим лёгкий инструмент для работы с кодом и текстовыми документами без избыточного функционала.

02 — Возможности

Основные возможности

📁

Работа с файлами

Создание, открытие и сохранение файлов в различных форматах. Поддержка быстрого доступа к недавним документам и управление проектом.

✏️

Редактирование текста

Полнофункциональный текстовый редактор с поддержкой основных операций: копирование, вставка, отмена действий, выделение и форматирование.

🎨

Синтаксическая подсветка

Цветовое выделение синтаксических конструкций для Python, JavaScript, Java и других языков программирования. Повышает читаемость кода.

💡

Автодополнение

Интеллектуальные подсказки при вводе кода: ключевые слова, имена переменных, стандартные функции и пользовательские идентификаторы.

🌙

Темы оформления

Переключение между светлой и тёмной темой. Поддержка пользовательских цветовых схем для комфортной работы в любое время суток.

🔍

Поиск по документу

Быстрый поиск и замена текста с поддержкой регулярных выражений. Навигация по совпадениям и подсветка найденных фрагментов.

🗂️

Вкладки

Одновременная работа с несколькими файлами через систему вкладок. Удобное переключение между документами без потери контекста.

💾

Автосохранение

Автоматическое сохранение изменений через заданные промежутки времени. Восстановление несохранённых данных после непредвиденного завершения работы.

07 — Этапы

Этапы разработки

Этап 1

Проектирование модулей

Определение компонентного состава системы, проектирование интерфейсов взаимодействия между модулями, создание диаграмм архитектуры.

Этап 2

Создание структуры репозитория

Инициализация репозитория Git, формирование файловой структуры проекта, настройка .gitignore и README.

Этап 3

Работа с ветками Git

Организация параллельной разработки через ветки: feature-ветки для каждого модуля, pull-запросы и code review перед слиянием.

Этап 4

Восстановление файлов из ревизий

Практика использования команд git log, git checkout и git revert для восстановления предыдущих версий файлов и отмены изменений.

Этап 5

Документирование кода

Написание docstring для всех публичных классов, методов и функций. Генерация HTML-документации с помощью автоматических инструментов.

Этап 6

Создание пользовательского руководства

Разработка инструкции по установке, запуску и использованию программы. Описание всех доступных функций и параметров настройки.

Этап 7

Разработка веб-сайта проекта

Создание одностраничного презентационного сайта на HTML и CSS для публикации на GitHub Pages. Оформление публичной страницы репозитория.

09 — Заключение

Заключение

В ходе выполнения лабораторной работы был спроектирован и реализован SmartNotepad — текстовый редактор с поддержкой семантики языков программирования. Проект охватывает полный цикл разработки: от проектирования архитектуры и работы с системой контроля версий до документирования и публичной презентации результатов.

Модульный подход позволил чётко разграничить зоны ответственности компонентов и создать расширяемую основу для дальнейшего развития продукта. Применение Git обеспечило прозрачную историю изменений и поддержку командной работы.

Проект наглядно демонстрирует применение принципов профессиональной программной инженерии на практике.
